VARIABLES

A variable is any characteristic, number or quantity that can be measured or counted.

Types of variables

  • Numerical
  • Categorical
  • Date and time
  • Mixed

Numerical Variables

A numerical variable is a variable where the measurement or number has a numerical meaning.

Its is two types


Discreate

The value which can be counted as whole value(fixed) called discreate variable

EXAMPLE -> 1, 45 not 23.5 , 56.75

Note - Only Integer values

Continous

The value contain range, measurement called as continous variable.

Example -> 1.3, 4.5, 20.3 not 23 45.

Note Only Floating values

Categorical Variable

The value of a categorical variable are selected from a group of categories also called labels.

Its isn of two types

Ordinal Variables

Ordinal variables are categorical variable in which the categories can be meaningfully ordered.

When the ordinal variable convert into the number then it has mathematical value.

Example -> Grades.

Nominal Variable

Nominal variable is same like ordinal variable but there is nothing that indicates an intrinsic order of the labels and in principle.

All labels are equal means there is no order.

No mathematical value

Example - colour

Date and Time


Date and Time variable can contain date only, time only, or date nad time.

Mixes Variable


The variable which conatains number and categories called as Mixed variable.
