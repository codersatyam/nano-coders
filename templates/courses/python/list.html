LIST


  • A list is a kind of collection

  • A collection allows us to put many values of same or different types in a single variavle.

  • A collection is good because we can carry many values around in one convenient package.

LIST CONSTANTS

  • List constants are surrounded by square brackets and the elements in the list are seperated by commas.

  • A list element cna be any pyton object, ever another list.

  • A list can be empty.

    • Example

    >>>print([1,24,76])
    [1,24,76]
    >>>print(['red','yellow','green'])
    ['red','yellow','green']
    >>>print(['red',1,2])
    ['red',1,2]
    >>>print([1,[5,6],7])
    [1,[5,6],7]
    >>>print([])

    Different operations of list


    1. append

    To add a new value in a list.
    The list stays in order and new element are added at the end of the list.

    Syntax -

    list.append(element)

    Parameter Values

    Parameter Description
    element element which is added to the list

    Example

    nums=[1,2,3,4]
    nums.append(6)
    print(nums)

    Output

    [1,2,3,4,6]


    2. insert()

    To add value but different from append that we can add at any place in a list using insert().

    Syntax -

    list.insert(position,element)

    Parameter Values

    Parameter Description
    position Required. A number specifying in which position to insert the value
    element Required. An element of any type (string, number, object etc.)

    Example

    nums=[1,2,3,4]
    nums.insert(2,6)
    print(nums)

    Output

    [1,2,6,3,4]


    3. clear()

    This function is use to remove all the elements from the list.

    Syntax -

    list.clear()

    Parameter Values


    NO parameter require

    Example

    nums=[1,2,3,4]
    nums.clear()
    print(nums)

    Output

    []


    4. remove()

    This function is use to remove the specific element from the list.

    Syntax -

    list.remove(element)

    Parameter Values

    Parameter Description
    element Required. Any type (string, number, list etc.) The element you want to remove

    Example

    nums=[1,5,2,3,4]
    nums.remove(2)
    print(nums)

    Output

    [1,5,3,4]


    5. pop()

    This function removes the element at the specified position.

    Syntax -

    list.remove(position)

    Parameter Values

    Parameter Description
    position Optional. A number specifying the position of the element you want to remove, default value is -1, which returns the last item

    Example 1

    nums=[1,5,2,3,4]
    nums.pop(1)
    print(nums)

    Output

    [1,2,3,4]


    Example 2

    nums=[1,5,2,3,4]
    nums.pop()
    print(nums)

    Output

    [1,5,2,3]


    6. del()

    This function is use to delete a group of elements.

    Syntax -

    del list[start:count]

    Parameter Values

    Parameter Description
    start Required. starting position from which elements are delete
    count Required. how many elements you want to delete from that position

    Example

    nums=[1,5,2,3,6,7,4]
    del nums[2:2]
    print(nums)

    Output

    [1,5,6,7,4]
