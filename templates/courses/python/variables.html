VARIABLES


What is Data Preprocessing ?


A variable is basically a container.

A variable is a named place in the memory where a programmer can store data and later retrieve the data using the variable "name"

Programmers get to choose the names of the variables.

You can change the content of a variable in a later statement.

Python variables name rule

  • Must start with a letter or underscores.
  • Must consist of lett6ers, numbers and underscores.
  • Case sensitive

Example


Right Example - nanocoder, nanocoder10, _nanocoder
Wrong Example - 10nanocoder, #nanocoder, nanocoder.10

Reserved Words


You can not use reserved words as variable name/identifiers.
