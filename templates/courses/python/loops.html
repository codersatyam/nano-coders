LOOPS


Use for repeate a particular piece of statement in our program.

1. while loop


While loop for certain reteration or conditions

In while loop we require 'indentation'.

Steps

1. Take a conter or variable
2. Check the given condition
3. Increment the value

Syntax -

while condition:

Example

i=1
while i<=5:
     print('Nanocoder')
     i=i+1

Output

Nanocoder
Nanocoder
Nanocoder
Nanocoder
Nanocoder


Nested while loop


Nested loop means a loop statement inside another loop statement.

For nested while loop we require different variables for every loop.

Example

i=1
while i<=3:
     print('Nanocoder',end=" ")
     j=1
     while j<=2:
          print("rocks",end=" ")
          j=j+1
     i=i+1
     print()

Output

Nanocoder rocks rocks
Nanocoder rocks rocks
Nanocoder rocks rocks


Some key points


We use j variable before starting inner loop because every time we want j start from 1 when comes out from inner loop.

We use 'end' so that ineer loop come in same line .

We use'end' in ineer loop so that all rocks come in same line .

In last we use print() so, that after completing one outer loop we come in next line.

2. for loop


for loop work with sequence

A for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string).

Example 1

x=['Nanocoder',65,2.5]
for i in x:
     print(i)

Output

Nanocoder
65
2.5


Some key points


In this i is a iterable variable.

i variable refers to a each element in a list or in a string.

In place of x we can put directly list, string,tuple,set.

3. for else


We use for else loop when our for loop will not give any output then else part will execute which shows that our for loop will not giv eany output .

Example

x=[12,16,18,21,26,56]
for i in x:
     if i%5==0:
          print(i)
else:
     print('Not Found')

Output

BREAK , CONTINUE and PASS


1. break


We use break to just come out of the loop.

Example 1

x=int(input('Enter the no of candies'))
i=1
while i<=x:
     print('candy')
     i=i+1

Output

Enter the no of candies 5
candy
candy
candy
candy
candy

Note- In the above example user can take any no of candy. But now the situation is come that we have a fix no of candy but user enter more candy to the available then we use 'break'

Example 2

available=3
x=int(input('Enter the no of candies'))
i=1
while i<=x:
     if i>available:
          break
     print('candy')
     i=i+1
print('By')

2. continue


It is not come out to the loop .It is just skip the value or statement.

Example

x=[1,2,3,4,5,6,7]
for i in x:
     if i%2==0:
          continue
     print(i)

Output

1
3
5
7
